Сравнение комплектаций

Вставьте два перечня позиций (одна позиция на строку, например «Стулья для класса x240») — расхождения по составу и количеству подсветятся ниже. Чисто построчное сравнение текста, без обращения к внешним сервисам.

Проверка полноты спецификации

Вставьте перечень позиций — проверка по ключевым словам отметит отсутствующие крупные категории (техника / мебель / дидактика), позиции без распознанного количества и повторяющиеся наименования. Это эвристика по ключевым словам, а не нормативная проверка по ФГОС/СанПиН.
